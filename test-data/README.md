# Test Data Folder

This folder contains test GEDCOM files and sample data for developing and testing the Family Tree application.

## Structure

- `gedcom/` - Sample GEDCOM files for testing import functionality
- `photos/` - Sample family photos for testing photo upload/display
- `json/` - Exported JSON files from the app for reference

## Usage

These files are meant for development and testing purposes only. They will be version-controlled in GitHub so you can easily reuse them across different environments.

## Adding New Test Data

When adding new test files:
1. Place them in the appropriate subfolder
2. Add a brief description here
3. Commit to GitHub for future use
